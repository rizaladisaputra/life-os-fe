# 📱 LifeOS

LifeOS adalah aplikasi mobile personal productivity yang membantu pengguna membangun kehidupan yang seimbang antara ibadah, kesehatan, karier, keuangan, dan hubungan sosial.

---

## ✨ Features

- 📅 Daily Planner
- ✅ Habit Tracker
- 🕌 Prayer Tracker
- 🇬🇧 English Learning
- 💼 Career Development
- 💰 Finance Dashboard
- 📖 Reading Tracker
- 📈 Statistics
- 🏆 Achievement & Gamification
- 🔔 Smart Reminder
- 🌙 Dark Mode

# 🛠 Tech Stack

## Mobile

- Flutter

- Spring Boot 3

# 🚀 Getting Started

Clone repository

```bash
git clone https://github.com/rizaladisaputra/life-os-fe.git
```

Masuk ke project

```bash
cd life-os-fe
```

Install dependency

```bash
flutter pub get
```

Run application

```bash
flutter run
```
