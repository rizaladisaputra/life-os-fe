# 📱 LifeOS

LifeOS adalah aplikasi mobile personal productivity yang membantu pengguna membangun kehidupan yang seimbang antara ibadah, kesehatan, karier, keuangan, dan hubungan sosial.

---

## ✨ Features

- 🔐 Authentication (Login & Signup with Email)
- 📅 Daily Planner
- ✅ Habit Tracker
- 🕌 Prayer Tracker
- 🇬🇧 English Learning
- 💼 Career Development
- 💰 Finance Dashboard
- 📖 Reading Tracker
- 📈 Statistics
- 🏆 Achievement & Gamification
- 🔔 Smart Reminder
- 🌙 Dark Mode

# 🛠 Tech Stack

## Mobile

- Flutter
- Dart
- Riverpod
- GoRouter
- Isar Database
- SharedPreferences (Auth Persistence)
- flutter_svg (SVG Support)

## Backend

- Spring Boot 3
- Java 21
- PostgreSQL
- JPA / Hibernate
- JWT Authentication
- Flyway

# 🚀 Getting Started

Clone repository

```bash
git clone https://github.com/rizaladisaputra/life-os-fe.git
```

Masuk ke project

```bash
cd life-os-fe
```

Install dependency

```bash
flutter pub get
```

Run application

```bash
flutter run
```
